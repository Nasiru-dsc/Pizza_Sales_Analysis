# Pizza Sales Analysis Project
### TechCrush Data Science - Task 3

This project analyzes a dataset of over **48,620 rows** to identify business KPIs and sales trends.

#### Key Performance Indicators (KPIs):
* **Total Revenue:** $817,860.05
* **Total Orders:** 21,350
* **Busiest Day:** Friday (3,538 orders)
* **Peak Month:** July (1,935 orders)

#### Technical Process:
* Integrated 4 datasets into a single master dataframe.
* Cleaned date data to resolve AttributeErrors for trend extraction.
* Automated KPI calculations using Python and Pandas.
